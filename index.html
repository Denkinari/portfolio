Experiência

HTML

70%

HTML é uma linguagem de marcação utilizada para criar e estruturar conteúdo na web. É a base de todas as páginas web e permite adicionar elementos como texto, imagens, vídeos, links e formulários, além de definir a estrutura do documento. O HTML é interpretado pelo navegador para exibir as páginas da web de acordo com o código fornecido. É uma das principais linguagens de programação para web e é uma parte essencial do desenvolvimento de sites e aplicativos web.

CSS

70%

CSS é uma linguagem de estilo usada para definir a aparência visual de páginas web escritas em HTML ou XHTML. Ele permite que você altere a cor, o tamanho, o layout e a fonte de texto, bem como outras propriedades visuais.

Javascript

30%

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, utilizada principalmente em aplicações web para adicionar interatividade e dinamicidade às páginas. É amplamente utilizado em frontend (interação com o usuário) e backend (processamento do lado do servidor) e é compatível com vários navegadores e sistemas operacionais. É uma linguagem de script, o que significa que o código é executado linha por linha no navegador ou servidor.

Photoshop

40%

O Photoshop é um software de edição de imagem e design gráfico desenvolvido pela Adobe. Ele é amplamente utilizado por profissionais de design, fotógrafos e artistas digitais para criar e editar imagens, além de criar gráficos, ilustrações, e outras mídias digitais.

Illustrator

30%

Adobe Illustrator é um software de edição de gráficos vetoriais usado para criar e editar gráficos vetoriais, desenhos, logotipos e ilustrações. Ele é usado principalmente por designers gráficos, artistas digitais, ilustradores e profissionais de marketing para criar gráficos e layouts para impressão, web e aplicativos móveis.

Figma

70%

O Figma é uma ferramenta de design gráfico e prototipagem de interfaces de usuário baseada na web. Ele permite que equipes colaborem em tempo real em projetos de design, compartilhem ativos e iterem em designs rapidamente. O Figma também oferece recursos de animação e design responsivo.

Referências Artísticas

Introduzindo então um pouco da minha vida, quando se trata de gosto musical e artístico, o meu tem tem uma forte influência a cultura japonesa atual. Sou eclético, mas tenho uma preferência por Indie-rock, Rock, Jazz, Pop e Eletrônica experimental, principalmente quando são bandas e/ou artistas japoneses. Também gosto de ouvir trilhas sonoras (OST) de jogos e animes enquanto trabalho. Alguns dos meus compositores favoritos são: Hiroyuki Sawano, Yuki Hayashi, Shoji Meguro e Akira Yamaoka.

Quanto aos conteúdos que gosto, geralmente me interesso em gêneros de suspense, mistério, terror psicológico e drama, seja em filmes, livros ou séries. São temas que me envolvem e me fazem ficar imerso. Agora no que diz respeito aos jogos, sou um grande fã de J-RPGs como Persona e Shin Megami Tensei, mas meu gosto costuma ser bem variado. Entretanto costumam ter minha atenção jogos que focam na narrativa e no desenvolvimento dos personagens, como indies e alguns títulos como Danganronpa, The House in Fata Morgana, The Last of Us e Nier: Automata.

Por fim, tenho tantos artistas e ilustradores que gosto, que fica dificíl até de citar, principalmente os que produzem mangás e compartilham suas artes independentes na internet. Alguns exemplos incluem Shigenori Soejima, Hayao Miyazaki e Ayami Kojima.

Bom, essas são algumas das minhas paixões e gostos pessoais. Espero que você tenha gostado de conhecer um pouco mais sobre mim. Ficarei feliz em compartilhar mais experiências e conhecimentos com você no futuro ou apenas trabalhar com você. :)
